# Simple File Storage Website

This is a simple web application that allows users to **upload**, **list**, and **download** files. It includes a **frontend** built with HTML, CSS, and JavaScript, and a **backend** powered by Python Flask.

---

## Features

- User-friendly file upload
- List all uploaded files
- Download files
- Clear error messages for upload issues
- CORS enabled for frontend-backend communication

---

## Technologies Used

- **Backend:** Python, Flask, Flask-CORS
- **Frontend:** HTML, CSS, JavaScript
- **Tools:** VS Code, Live Server (for frontend)

---
